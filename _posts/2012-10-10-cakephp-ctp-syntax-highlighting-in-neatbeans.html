  CAKEPHP (.CTP) SYNTAX HIGHLIGHTING IN NETBEANS  

 

 

To enable syntax highlight for .ctp files of CakePHP in NetBeans you should follow the following steps:

 

Tools -> Options -> Miscellaneous -> Files

New File Extension (ctp)

Associate with PHP

Now your .CTP files will be treated as PHP file in NetBeans. Happy coding!!
