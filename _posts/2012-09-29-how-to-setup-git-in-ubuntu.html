?HOW TO SETUP GIT IN UBUNTU?

GIT is one of the most useful stuff to work n a multi users application development. To install GIT?on UBUNTU?you need to follow few steps like below:

  • Install GIT Core in your local UBUNTU machine 
    sudo apt-get install git-core
  • Verify the installation was successful by typing the following comman 
    $which git
/opt/local/bin/git

     

  • Configure your username using the following command: 
    git config --global user.name "FIRST_NAME LAST_NAME"

     

  • Configure your email address using the following command: 
    git config --global user.email "MY_NAME@example.com"

     

Now you have successfully configured your GIT. Now have a happy coding!!
